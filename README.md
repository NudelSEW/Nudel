# nudel
Nudel - Date Reserving Tool

Dieses Repository soll alle Dateien und Dokumente für das Projekt "Terminreservierungssoftware" enthalten.
Es dient als Schnittstelle zwischen den einzelnen Mitgliedern.

## Funktionsweise

Die Desktop-Application "Nudel" lässt einen User mittels Internet-Verbindung auf einen Nudel-Server zugreifen, um verschiedene Dinge zu ermöglichen.
Alle neuen Benutzer müssen sich registrieren und vorhandene können sich einloggen.
